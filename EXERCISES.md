# 🛠️ 練習課題集 — Claude Code でアプリを育てよう

このアプリは「**動く土台**」です。ここから **Claude Code といっしょに**、自分だけのアプリに育てていきましょう。
むずかしいコードは書けなくて大丈夫。**やりたいことを日本語でお願いする**のが、このアプリの作り方です。

> はじめる前に、Claude Code にこう言ってみてください👇
> 「`CLAUDE.md` と `EXERCISES.md` を読んで。まずアプリをローカルで起動して、ブラウザで見せて」

## 進め方の3つのコツ
1. **小さく頼む**：一度に1つだけ。「◯◯を△△に変えて」くらいの粒度がうまくいきます。
2. **すぐ確認する**：変えたらブラウザを更新して、自分の目で見る（スマホ表示がおすすめ）。
3. **こわれても大丈夫**：「さっきの変更を元に戻して」と言えば戻せます。気軽に試そう。

---

## 🌱 レベル1：文章を自分の言葉にする（いちばんやさしい）
さわるファイルは基本 `js/data.js` だけ。コードは触りません。

- **1-1 タスクを増やす／書き換える**
  - 頼み方例：「`data.js` の今日の5分タスクに『郵便受けを空にする』を追加して」
- **1-2 応援メッセージを自分らしく**
  - 頼み方例：「応援メッセージを、私の受講生さんに響く言葉に5つ書き換えて」
- **1-3 お部屋の成長メッセージを変える**
  - 頼み方例：「`roomStages` の各レベルの言葉を、もっとワクワクする表現にして」

## 🎨 レベル2：見た目を変える
- **2-1 テーマカラーを変える**
  - 頼み方例：「テーマカラーを、やさしい水色ベースに変えて」（`css/styles.css` の `:root`）
- **2-2 アプリの名前・キャッチを変える**
  - 頼み方例：「アプリ名とキャッチコピーを『◯◯』に変えて」（`js/data.js` と `index.html` のタイトル）

## 🎁 レベル3：新しい機能を足す（ここが本番！）
※ これらは“あえて未実装”です。Claude Code と作ってみましょう。

  - 頼み方例：「できた回数に応じて集まる『ごほうびシール帳』を新しく作って」

  - 頼み方例：「『グッズ』タブを追加して、おすすめ収納グッズを画像とひとことで紹介できるようにして。場所診断の結果と連動させて」

- **3-4 続けたくなるリマインド**

## ✨ 自由制作のヒント
「こんなのあったらいいな」を、そのまま Claude Code に話してOK。例：
- 「今日の気分を記録できるようにして、グラフで見たい」
- 「家族みんなで使えるように、メンバーを切り替えられるようにして」
- 「達成したらレベルが上がる『片付けレベル』を表示して」

うまく動かないときは、**そのままエラーや画面の様子を Claude Code に伝える**と直してくれます。
